## Privacy Policy for 60FPS Upload Manager

Last Updated: December 2025

1. Data Collection
We collect an anonymous Device ID to manage software licenses and prevent abuse. We may collect basic usage logs (e.g., "Installation", "Update") to improve system stability.

2. User Information
We retrieve public profile information (username, avatar) only to display it within the extension's interface. This data is not sold to third parties.

3. Permissions
- "Storage": Used to save your theme preferences and Device ID.
- "Alarms": Used to periodically check for critical updates.
- "Downloads": Used to save media files to your computer upon your request.

4. Contact
For any questions, please contact the developer via the support tab.
